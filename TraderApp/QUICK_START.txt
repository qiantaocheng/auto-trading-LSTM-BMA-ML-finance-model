Connection Won't Work?
  1. Verify TWS/IB Gateway is running
  2. Check Mode is set correctly
  3. Verify Client ID is valid (0-9999)
  4. Ensure port 4002 (Paper) or 4001 (Live) is open
  5. Check TWS API settings allow connections

Client ID Won't Update?
  1. Must be disconnected to change
  2. Click [Disconnect] first
  3. Then edit Client ID
  4. Click [Connect] to apply

No Capital Data?
  1. Must be connected first
  2. Wait 1 minute for first sample
  3. Check database file exists
  4. Restart app if needed

Auto Trading Not Triggering?
  1. Verify [âœ“] Enabled is checked
  2. Run a prediction first
  3. Check HMM status (not crisis)
  4. Monitor risk gate value
  5. Check trading log for errors
